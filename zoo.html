Contexte du projet

Projet réalisé en BTS CIEL — l'objectif était de créer un site web pour le Zooparc de Thoiry en HTML/CSS, puis de le transformer dynamiquement via JavaScript pour le faire ressembler au Zoo de Beauval : changement du logo, du titre, de la couleur du header, des liens réseaux sociaux, et enrichissement de la galerie d'animaux.

1 · Page d'accueil

Accueil — Zooparc de Thoiry transformé en Beauval via JS Le script modifie dynamiquement le titre, le logo et la couleur du header au chargement. La navigation latérale mène vers les pages Animaux, Accès, Actualités et Réservations.

2 · Page animaux

Galerie animaux — tableau HTML enrichi dynamiquement La galerie affiche 20 animaux en grille 4 colonnes. Le JS ajoute une ligne complète (Saimiri, Colibri, Renard, Gnou), remplace des images, colore les noms en jaune, et ajoute une légende via caption.

4 · Extraits de code

script.js — transformation dynamique Thoiry → Beauval 
// Transformation du thème au chargement
document.querySelector("h1").innerHTML = "ZOOPARC DE BEAUVAL";
document.getElementById("logo").setAttribute("src", "Images/logo2.png");
document.querySelector("header").style.backgroundColor = "#006BB1";

// Mise à jour des liens réseaux sociaux
list = document.querySelectorAll("a");
list[6].setAttribute("href", "https://www.facebook.com/zoobeauval");
list[7].setAttribute("href", "https://twitter.com/zoobeauval/");
list[8].setAttribute("href", "https://www.instagram.com/zoobeauval/");
script1.js — ajout d'une ligne au tableau + événements interactifs 
// Création dynamique d'une ligne avec 4 nouveaux animaux
let newtr = document.createElement("tr");
let tdnew = document.createElement("td");
let imgnew = document.createElement("img");
imgnew.setAttribute("src", "Images/colibri.jpg");
tdnew.appendChild(imgnew);
newtr.appendChild(tdnew);
document.querySelector("table").appendChild(newtr);

// Interaction : clic sur le header → couleur aléatoire
document.querySelector("header").addEventListener("click", function() {
    this.style.backgroundColor =
        '#' + (Math.random() * 0xFFFFFF << 0).toString(16);
});
